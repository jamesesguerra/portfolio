LATEST PROJECT

kitchenet: Github for food recipes

Think of this as Github for food recipes, where you can share, discover, and collaborate on your favorite dishes. Users can suggest changes to your recipes, and you can review them in the same way you would review code changes.

The app is built using Angular paired with .NET. For storage, I chose MinIO to manage images and PostgreSQL to store the rest of the recipe and user data. As I don’t have the expertise (or the patience) to build custom authentication, I opted for Auth0 to handle it for me.
